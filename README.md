A terminal-based **Rock-Paper-Scissors** game built using Python. This project is a classic implementation where the player competes against the computer in a race to score 3 points.

## 🚀 Features
* **Interactive Gameplay:** Play directly inside your command line or terminal.
* **Point System:** First to reach **3 points** wins the match.
* **Smart Computer Opponent:** Uses randomized decision-making for the computer's moves.
* **Input Validation:** Simple integer-based selection (`1` for Rock, `2` for Paper, `3` for Scissors).

---

## 🛠️ Built With (Libraries & Tools)
* **Python 3.x** - Core programming language.
* **`random` (Built-in)** - Python's standard library module used to generate randomized choices for the computer. *No external installations or `pip install` commands are needed!*

## 💻 How to Clone and Run the Project

Follow these simple steps to download and run the game on your local machine:

### 1. Clone the Repository
Open your terminal or command prompt and run the following command to download the code:
```bash
git clone https://github.com/saqib783/rock-paper-scissors-.git
```

### 2. Navigate to the Directory
Go into the project folder:
```bash
cd rock-paper-scissors-
```

### 3. Run the Game
Execute the Python file using your terminal:
```bash
python game.py
```

---

## 🎮 How to Play
1. Once the game starts, you will see a prompt asking for your choice.
2. Enter:
   * `1` for **Rock**
   * `2` for **Paper**
   * `3` for **Scissors**
3. The computer will automatically generate its choice.
4. The terminal will display who won that round and keep track of the scores.
5. The game ends automatically as soon as either you or the computer reaches **3 points**.
